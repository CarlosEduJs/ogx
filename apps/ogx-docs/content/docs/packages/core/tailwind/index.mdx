OGX supports a subset of Tailwind CSS utilities optimized for image generation.

## Parser v2

OGX v0.2.0 uses a high-performance parser with O(1) static class lookups and multi-level caching. The parser handles ~170 static classes via Map-based lookup tables and processes dynamic classes through modular prefix handlers.

## Layout

| Class | Description |
|-------|-------------|
| `flex`, `flex-row`, `flex-col` | Flex direction |
| `items-*`, `justify-*` | Alignment |
| `gap-{n}` | Gap between children |
| `grow`, `shrink`, `shrink-0` | Flex grow/shrink |

## Spacing

| Class | Description |
|-------|-------------|
| `p-{n}`, `px-{n}`, `py-{n}` | Padding |
| `pt-{n}`, `pr-{n}`, `pb-{n}`, `pl-{n}` | Directional padding |
| `m-{n}`, `mx-{n}`, `my-{n}` | Margin |
| `mt-{n}`, `mr-{n}`, `mb-{n}`, `ml-{n}` | Directional margin |

## Sizing

| Class | Description |
|-------|-------------|
| `w-{n}`, `w-full`, `w-auto`, `w-fit` | Width |
| `h-{n}`, `h-full`, `h-auto`, `h-fit` | Height |
| `max-w-{n}`, `min-w-{n}` | Min/max width |

## Typography

| Class | Description |
|-------|-------------|
| `text-{size}` | Font size (xs to 9xl) |
| `font-{weight}` | Font weight (light to black) |
| `text-{color}` | Text color |
| `tracking-{val}` | Letter spacing |
| `leading-{val}` | Line height |
| `text-center`, `text-left`, `text-right` | Alignment |
| `truncate` | Overflow ellipsis |

## Backgrounds

| Class | Description |
|-------|-------------|
| `bg-{color}` | Background color |
| `bg-gradient-to-{dir}` | Gradient direction |
| `from-{color}`, `via-{color}`, `to-{color}` | Gradient colors |

## Borders

| Class | Description |
|-------|-------------|
| `border`, `border-{n}` | Border width |
| `border-{color}` | Border color |
| `rounded-{size}` | Border radius |

## Effects

| Class | Description |
|-------|-------------|
| `shadow-{size}` | Box shadow |
| `opacity-{n}` | Opacity |
| `backdrop-blur-{size}` | Backdrop blur |
