## Font Size

| Class | Size | Line Height |
|-------|------|-------------|
| `text-xs` | 12px | 16px |
| `text-sm` | 14px | 20px |
| `text-base` | 16px | 24px |
| `text-lg` | 18px | 28px |
| `text-xl` | 20px | 28px |
| `text-2xl` | 24px | 32px |
| `text-3xl` | 30px | 36px |
| `text-4xl` | 36px | 40px |
| `text-5xl` | 48px | 48px |
| `text-6xl` | 60px | 60px |
| `text-7xl` | 72px | 72px |
| `text-8xl` | 96px | 96px |
| `text-9xl` | 128px | 128px |

```typescript
'text-6xl'       // 60px font
'text-[42px]'    // Arbitrary 42px
```

| Class | Value |
|-------|-------|

## Example: Heading Styles

```typescript
// Large bold heading
'text-6xl font-bold tracking-tight'

// Subtitle
'text-xl text-zinc-400 leading-relaxed'

// Uppercase label
'text-sm font-medium uppercase tracking-widest'
```
