The docs preset is designed for documentation sites with clear hierarchy and branding.

## Usage

```typescript
const png = await ogx({
  preset: 'docs',
  title: 'Getting Started with OGX',
  description: 'Learn how to generate OG images',
  siteName: 'OGX Docs',
  colorScheme: 'dark',
  logo: logoBase64,
});
```

## Options

| Option | Type | Required | Description |
|--------|------|----------|-------------|
| `title` | `string` | Yes | Main heading |
| `description` | `string` | No | Subtitle text |
| `siteName` | `string` | No | Site name in header |
| `logo` | `string` | No | Logo URL or base64 |
| `colorScheme` | `'dark' \| 'light'` | No | Color theme (default: dark) |
| `detailsOG` | `boolean` | No | Show detailed background pattern |

## Features

- Clean header with logo and site name
- Documentation badge indicator
- Large title with tight tracking and max-width control
- Description with muted color
- Indicator bar footer
- Light and dark color scheme support
- Fast rendering (~47ms average)
- Optional detailed background pattern for visual interest
